✨ Feature Section – Frontend Practice (Day 05)

A clean and modern Feature Section UI component built using HTML & CSS as part of my Frontend Development Practice Series.

This project focuses on practicing card-based layouts, grid systems, and responsive UI design, which are common patterns used in modern landing pages.

✔ Clean feature section layout

✔ Three feature cards with icons

✔ Hover micro-interactions on cards

✔ Responsive grid layout

✔ Proper spacing and visual hierarchy

✔ Modern card UI design

✔ Mobile-friendly layout for smaller screens
